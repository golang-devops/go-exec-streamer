# go-exec-streamer
A golang exec streamer to make streaming exec.Command as simple as possible

## Usage

### Windows example

main.go:

```
package main

import (
    execstreamer "github.com/golang-devops/go-exec-streamer"
    "log"
    "os"
)

func main() {
    streamer, err := execstreamer.NewExecStreamerBuilder().
        ExecutorName("winshell").
        Exe("ping").
        Args("127.0.0.1", "-n", "6").
        StdoutWriter(os.Stdout).
        StderrWriter(os.Stderr).
        StdoutPrefix("OUT:").
        StderrPrefix("ERR:").
        AutoFlush().
        Build()

    if err != nil {
        log.Fatal(err)
    }

    err = streamer.ExecAndWait()
    if err != nil {
        log.Fatal(err)
    }
}
```

Note: `StdoutWriter` and `StderrWriter` is set to the `ioutil.Discard` writer by default.

Now run it:

```
go run main.go
```
